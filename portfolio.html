My Portfolio

"Satisfying To-do List" - Android Mobile Application

The purpose of this project was to create a an application that essentially hacks the reward center of the brain in order to complete simple mundane tasks, it does this by playing a rewarding sound effect that causes a small dopamine spike in the brain each time a task is completed. When all tasks are completed, all tasks are displayed green which creates an effect that some users may find satisfying.

This app was designed to be incredible simple and easy to use, when no tasks are completed, there is no colour on screen, this means users are required to complete their daily tasks in order to add colour to the screen.

Coming soon on the Google Play Store.

"Cold War Barracks" - Android Mobile Application

This app was created to allow fans of the upcoming Call of Duty game to see stats and information about in game items, before the game is released, it served as the only place to see the information.

It was designed to use a clean and intuitive layout, to make it as simple as possible to see and find the information displayed. Created in Android Studio, written in Java.

"Food Friend" - Android Mobile Application

Find and meet people that share a common interest in food. Coded in Android Studio (Java) for my university final year project. It offers a fully functioning account and login system, with the ability to discover other users and send messages to them, users can also upload and crop local images all within the app, "Google Place Picker" is also implemented to allow users to find locations on Google Maps. Firebase was used for the backend of this project to store all user data.
