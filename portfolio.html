Portfolio

"Food Friend" - Android Mobile Application

Find and meet people that share a common interest in food. Coded in Android Studio for my university final year project and continued after graduating. Firebase is used for the server side/database.

"Connect 4" - HTML/Javascript

Fully working Connect 4 game played in any browser, play 1v1 locally or online, or play against an AI. Created using HTML, CSS, Javascript, Node.js and Socket.io.

"OpenChat" - Android Kotlin App

Group instant messaging app, coded in Kotlin with Firebase database for server side, this project was done to learn Kotline and simply to demonstrate what I have learnt.

"Awoken" - Unity Game / 3ds Max Scene

Item finding game created in Unity. All assets/models created by me in 3ds Max and Photoshop.

"Student Support Ticket" - Android App

App that allows students to request help from the tutor in the form of tickets, tutor can create a session for the class and remove tickets when help has been given. Created in Android Studio.

"Flip" - Unity Game

Unique minigame created in Unity. Move left, right and jump to avoid obstacles, jumping will flip the player, game over if bottom colour of player object does not match colour of the ground.

"CGD Showcase" - Android App
